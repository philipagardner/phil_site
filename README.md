# Phil Gardner Portfolio

A React portfolio site for Phil A. Gardner, built with Vite and React Router to showcase UX engineering, accessible front-end work, and agentic AI interface experience.

## Pages

- `/` - Home page
- `/work` - Selected work, delivery approach, and core strengths
- `/services` - Legacy route alias that points to the same work page
- `/about` - Background, working principles, and project stories
- `/contact` - Contact page with client-side form feedback
- `/game` - A playable Snake page included as a lightweight demo element

## Structure

- `src/components` - Shared layout and section components
- `src/pages` - Route-level pages
- `src/data` - Resume-inspired JSON content for feedback, strengths, and project stories
- `src/styles.css` - Shared site styling

## Local Development

This project needs Node.js and npm installed locally.

1. Install Node.js LTS.
2. Run `npm install`.
3. Run `npm run dev`.

If PowerShell blocks `npm`, use `npm.cmd install`, `npm.cmd run dev`, and `npm.cmd run build` instead.

Production output is generated in `dist`.
